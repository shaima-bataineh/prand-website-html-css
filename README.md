# Brand Website (Frontend Project)

A responsive multi-page website built using HTML and CSS, designed to present a brand with a clean and user-friendly interface.

## Features
- Multi-page website (Home, About, Services, Gallery, Contact)
- Responsive design for different screen sizes
- Clean and structured layout
- Simple login page UI

## Technologies Used
- HTML5
- CSS3 

## Project Structure
- index.html (Home page)
- about.html
- services.html
- gallery.html
- contact.html
- login.html
- css/
- img/
